# 📚 دليل هيكل المشروع

## 📁 Assets/

### Scripts/
- **Player/** - سكريبتات حركة اللاعب
  - `PlayerController.cs` - التحكم في الحركة والمدخلات

- **Game/** - مديرو نظام اللعبة
  - `GameManager.cs` - إدارة حالة اللعبة والمشاهد
  - `ScoreSystem.cs` - نظام النقاط
  - `LevelManager.cs` - إدارة المستويات والصعوبة

- **UI/** - واجهة المستخدم
  - `UIManager.cs` - إدارة جميع عناصر الواجهة
  - `MainMenuController.cs` - منطق القائمة الرئيسية

- **Collectibles/** - الأشياء المجمعة
  - `Collectible.cs` - سلوك الأشياء المجمعة

- **Hazards/** - المخاطر والأعداء
  - `Hazard.cs` - سلوك المخاطر والحركة

- **Audio/** - إدارة الصوت
  - `AudioManager.cs` - نظام تشغيل الأصوات

### Prefabs/
- جميع GameObjects المُعاد استخدامها

### Scenes/
- `MainMenu.unity` - القائمة الرئيسية
- `Level1.unity` - المستوى الأول (سهل)
- `Level2.unity` - المستوى الثاني (متوسط)
- `Level3.unity` - المستوى الثالث (صعب)

### Sprites/
- جميع الصور والرسومات

### Audio/
- جميع ملفات الصوت والموسيقى

---

## 🎮 آلية اللعبة

### الحركة
- WASD أو أسهم لوحة المفاتيح
- حركة سلسة مع Rigidbody2D

### جمع الأشياء
- اقترب من الأشياء الصفراء
- 10 نقاط لكل شيء
- صوت عند الجمع

### المخاطر
- تصادم مع المخاطر = فقدان حياة
- تأثير knockback
- صوت عند الإصابة

### نظام الحياة
- 3 أرواح في البداية
- Game Over عند 0 أرواح

### المؤقت
- يختلف حسب المستوى
- Game Over عند انتهاء الوقت

## 🔌 الاتصالات بين الأنظمة

```
GameManager
  ├── UIManager (عرض البيانات)
  ├── ScoreSystem (إدارة النقاط)
  ├── LevelManager (إدارة المستوى)
  └── PlayerController (حالة اللعبة)

PlayerController
  ├── GameManager (الحياة)
  ├── AudioManager (الأصوات)
  └── Hazard (الاصطدام)

Collectible
  ├── GameManager (جمع الشيء)
  └── LevelManager (عد الأشياء)

Hazard
  └── PlayerController (الضرر)
```
