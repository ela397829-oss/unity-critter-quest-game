# 🎮 Critter Quest - Unity 2D Game

لعبة جمع الأشياء 2D كاملة وجاهزة للعمل مباشرة في Unity Editor.

## 📋 محتويات اللعبة

- ✅ نظام حركة اللاعب
- ✅ نظام جمع الأشياء (Collectibles)
- ✅ نظام الأعداء والمخاطر (Hazards)
- ✅ نظام الأرواح (Lives)
- ✅ عداد الوقت (Timer)
- ✅ 3 مستويات صعوبة مختلفة
- ✅ واجهة المستخدم كاملة (UI)
- ✅ نظام الصوت (Audio)
- ✅ قائمة رئيسية (Main Menu)
- ✅ شاشة Game Over و Win

## 🚀 البدء السريع

### المتطلبات:
- Unity 2022.3 LTS أو أحدث
- TextMeshPro (سيتم تثبيته تلقائياً)

### خطوات التثبيت:
1. استنسخ المستودع:
```bash
git clone https://github.com/ela397829-oss/unity-critter-quest-game.git
```

2. افتح المشروع في Unity Hub
3. اتبع `SETUP_INSTRUCTIONS.md` للإعداد الكامل

## 📁 هيكل المشروع

```
Assets/
├── Scripts/
│   ├── Player/          # سكريبت حركة اللاعب
│   ├── Game/            # مديرو اللعبة
│   ├── UI/              # واجهة المستخدم
│   ├── Collectibles/    # سكريبت الأشياء المجمعة
│   ├── Hazards/         # سكريبت المخاطر والأعداء
│   └── Audio/           # مدير الصوت
├── Prefabs/             # Prefabs جاهزة
├── Scenes/              # جميع المستويات والقوائم
├── Sprites/             # الصور والرسومات
├── Audio/               # ملفات الصوت
└── Resources/           # موارد إضافية
```

## 🎮 طريقة اللعب

1. **الحركة**: استخدم WASD أو أسهم لوحة المفاتيح
2. **الجمع**: اقترب من الأشياء الصفراء لجمعها
3. **تجنب المخاطر**: ابتعد عن الأشياء الحمراء
4. **الهدف**: اجمع جميع الأشياء قبل انتهاء الوقت

## 📊 مستويات اللعبة

- **Level 1**: سهل (5 أشياء، 90 ثانية، 2 مخاطر)
- **Level 2**: متوسط (8 أشياء، 75 ثانية، 4 مخاطر)
- **Level 3**: صعب (10 أشياء، 60 ثانية، 6 مخاطر)
